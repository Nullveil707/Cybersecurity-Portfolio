# Botium Toys Internal Security Audit

## Project Summary

Conducted an internal security audit for a fictional retail company using the NIST Cybersecurity Framework (CSF).

## Skills Demonstrated

- Risk Assessment
- Security Controls Evaluation
- Compliance Analysis
- PCI DSS
- GDPR
- SOC Controls

## Key Findings

- Missing least privilege controls
- No intrusion detection system
- No disaster recovery plan
- Lack of encryption for customer payment data

## Recommendations

Recommended implementing least privilege, IDS, backups, disaster recovery planning, encryption, and centralized password management to improve the organization's security posture.
